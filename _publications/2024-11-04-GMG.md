* Open-source software package to jointly visualize different geophyiscal data sets in 3D
* Tools for prjection of geographical coordinates (Lon, Lat, Dep) to Cartesian coordinates (X, Y, Z)
* Tools to create 3D setups for numerical models (e.g., LaMEM)
